# 🎨 Background Changer Website

A simple, interactive, and responsive web application that changes the background color of the webpage dynamically. Built purely using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

*   **Dynamic Color Generation:** Changes the background color instantly upon clicking a button.
*   **Hex/Color Code Display:** Shows the current background color code so users can copy it.
*   **Clean UI/UX:** Minimalist, modern, and eye-catching design.
*   **Fully Responsive:** Works perfectly on desktops, tablets, and mobile devices.

---

## 🛠️ Tech Stack Used

*   **HTML5:** For creating the structure of the website.
*   **CSS3:** For styling, layouts, and smooth transitions.
*   **JavaScript (ES6):** For handling the click events and generating colors.
